# Data Structure
## ArrayList (List)
- with order
- allow duplicated object
- get(index)
- remove(Object)
- remove(index)
- addLast(Object)

## LinkedList (Deque/ Queue/ List)
- with order
- allow duplicated object
- remove head/tail (java LinkedList -> double LinkedList)
- remove(Object)
- addLast(Object)

## ArrayDeque (Deque/ Queue)
- with order
- allow duplicated object

## HashSet (set)
- No order
- disallow duplicated object

## HashMap (Map)
- No order
- Entry (Key + Value)
